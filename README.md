# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```

3. Tener Nest CLI instalado

```
npm i -g @nestjs/cli
```

4. Levantar la base de datos

```
docker-compose up -d
```

5. Clonar le archivo ```.env.template``` y renombrar la copia a ```.env```

6. Levantar el proyecto
```
yarn start:dev
```


## Stack usado

- MongoDB
- Nest
